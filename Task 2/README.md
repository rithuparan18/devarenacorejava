# Grade Management System - Java Console App

## Project Overview
This project is a Grade Management System built for Week 2 of my Java course. It demonstrates mastery of control flow (loops, switch statements), data collections using Arrays and `ArrayList`, and basic error handling using `try-catch` blocks. The system allows users to manage student marks, calculate averages, and automatically generate comprehensive performance reports.

## Features
* **Add Student Marks:** Input marks for multiple subjects stored securely in arrays.
* **View All Records:** Formatted table displaying all subject marks, averages, and grades.
* **Search:** Case-insensitive search to pull up specific student records.
* **Performance Reports:** Calculates class averages per subject, ranks top performers, and graphs grade distribution.
* **Input Validation:** Strict `try-catch` validation ensures marks are numeric and between 0-100 without crashing the program.

## Setup Instructions
1. Clone this repository and navigate to the `src` directory.
2. Ensure you have the **Java Development Kit (JDK)** installed.
3. Compile the Java files: `javac *.java`
4. Run the main application: `java GradeManagementSystem`

# Student Information System - Java Console App

## Project Overview
This project is a console-based Java application built for Week 1 of my Java Fundamentals course. It allows users to manage student records via a menu-driven interface. Users can Create, Read, Update, Delete, and Search (CRUD+S) student information seamlessly. 

## Features
* **Add Student:** Input a student's ID, Name, Age, Grade, and Contact Info.
* **View Students:** Displays all records in a formatted table.
* **Search:** Find a student quickly using their ID or Name.
* **Update/Delete:** Modify existing records or remove them.
* **Data Validation:** Ensures age is a positive number and grades are between 0 and 100.

## Setup Instructions
1. Ensure you have the **Java Development Kit (JDK)** installed on your machine.
2. Clone this repository to your local machine.
3. Open your terminal/command prompt and navigate to the `src` directory.
4. Compile the Java files: `javac *.java`
5. Run the main program: `java StudentInformationSystem`
